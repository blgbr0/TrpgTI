## 当前视觉方向

为了更贴近 `sbti.ai` 的图鉴质感，所有人格图统一采用：

1. `四字母代号` 作为主视觉锚点
2. `人格名` 作为二级识别
3. `跑团梗词` 作为海报中的大字元素
4. `统一边框、统一版式、不同配色`

这样用户一眼会觉得：

- 这是一个有分类系统的测试
- 每个类型都有自己独立的“卡面”
- 晒图时看起来像真正的结果海报，而不是文字截图
