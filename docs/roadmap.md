# 实施路线

## 1. 当前已完成

1. 核心定位与传播策略
2. 16 个正式人格定义
3. 24 题首版题库
4. 4 个隐藏人格与彩蛋题机制
5. 静态前端首版实现

## 2. 当前版本重点

首版重点不是“绝对准确”，而是验证三件事：

1. 用户是否愿意完整做完
2. 用户是否愿意认领结果
3. 用户是否愿意把结果转发给同车队的人

## 3. 建议验证流程

1. 找 5 到 10 名真实跑团玩家内测
2. 覆盖新手、老玩家、KP/DM
3. 每人至少做一次，不预先解释人格内容
4. 记录以下反馈：

- 哪个题读起来最别扭
- 哪个结果最像自己
- 哪个结果最像队友
- 会不会发到群里
- 会不会为了好玩再测一次

## 4. 关键观察指标

1. 完成率
2. 平均耗时
3. 分享率
4. 二次传播率
5. 结果认领率

## 5. 下一阶段实施项

1. 优化结果页视觉冲击
2. 加入复制分享文案
3. 加入 PNG 分享图导出
4. 根据内测反馈重写 20% 题面
5. 继续扩展第五个隐藏人格，完善彩蛋体系

## 6. 中期扩展方向

1. `COC` 专项版：偏调查、掉 san、信任危机、隐秘线索
2. `DND` 专项版：偏职业分工、阵营选择、小队协作
3. `KP/DM` 版：偏控场、放刀、即兴应变、玩家管理
4. `车队关系图`：支持多个玩家结果并排展示
