# KANG 杠精

## 类型定位

你不一定想反对，但你一定能找到一个值得反对的角度。

## 一句话暴击

`你不一定想反对，但你一定能找到反对角度。`

## 风格化长描述

你不一定想反对，但你一定能找到一个值得反对的角度。KP刚说完“理论上不行”，你已经准备好三个例外条款和两个前置判例。你不是来抬杠的，你是来测试系统耐久度的。

## 典型表现

- 一句“不行”会激活你寻找例外条款的本能。
- 你擅长发现主持人、规则和场景描述里的缝。
- 你不一定故意拆台，但你天然会逼系统自证完整性。

## 团内生态位

负责给每个本做一次压力测试。
