# TALK 嘴遁王

## 类型定位

能靠嘴拿下的场面，你连骰子都懒得摸。

## 一句话暴击

`能靠嘴拿下的场面，你连骰子都懒得摸。`

## 风格化长描述

你最大的武器不是刀不是枪，是那张能把敌我识别系统说到短路的嘴。别人把社交轮当过场，你能把过场聊成支线、把支线聊成交易、把交易聊成感情戏。你不是嘴硬，你是输出方式文科。

## 典型表现

- 你遇到NPC先寒暄，再套话，再谈条件。
- 你对沉默的容忍度低于对战斗的容忍度。
- 很多本来该打一架的场面，会被你聊成交易现场。

## 团内生态位

负责把社交回合打成持续输出回合。

## 高权重维度

- `TALK` 社交欲：高
- `SHOW` 表演欲：中高
- `PUSH` 推进欲：中
- `SUS` 多疑度：低

## 结果页分享文案

`我测出来是 TALK 嘴遁王，这团能不能活不知道，但我先把条件谈下来。`
