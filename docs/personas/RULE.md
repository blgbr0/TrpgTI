# RULE 律师

## 类型定位

你不是来听 KP 说书的，你是来查法条的。

## 一句话暴击

`你不是来听KP说书的，你是来查法条的。`

## 风格化长描述

别的玩家带骰子，你带法条。别人听完描述决定行动，你听完描述先判断主持人的措辞有没有漏洞。你不是爱抬杠，你只是无法容忍“口头约定高于规则文本”这种原始部落行为。

## 典型表现

- 一句“理论上不行”会瞬间点亮你的大脑。
- 你会本能地区分文本、口胡和 house rule。
- 你纠正流程时的表情，像在法庭上提交证据。

## 团内生态位

负责确保全桌至少在同一套规则里发疯。

## 高权重维度

- `RULE` 规则执念：高
- `PUSH` 推进欲：中高
- `SAFE` 保命欲：中
- `CHAOS` 整活欲：低

## 结果页分享文案

`我测出来是律师，今天这个判定要么合法，要么重审。`
