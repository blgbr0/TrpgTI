# HIDE 苟王

## 类型定位

你不是胆小，你只是比所有人都更早理解了角色卡是会死的。

## 一句话暴击

`活着才有输出，死了只剩复盘。`

## 风格化长描述

你不是胆小，你只是比所有人都更早理解了角色卡是会死的。队友一听门后有动静就上，你先问后门、窗户、掩体和撤退路线。你不是不给节目效果，你只是坚信活着的人才有资格复盘。

## 典型表现

- 你不是不冲，你只是要先知道冲完怎么活。
- 所有门、路、掩体和退路你都会先看一遍。
- 你可能不是最炸眼的那个，但常常是最后还站着的那个。

## 团内生态位

负责提醒全队：角色卡不是消耗品。

## 高权重维度

- `SAFE` 保命欲：高
- `SUS` 多疑度：中高
- `RISK` 作死欲：低
- `CHAOS` 整活欲：低
