# MBTI 插画风格拆解

这份文档基于对 `16Personalities` 官方 SVG 角色图的直接查看与结构拆解，目的是让我们的跑团人格图从“抽象海报”转向更接近 `MBTI` 的人物插画语言。

## 1. 最核心的结论

`16Personalities` 的图不是 logo 海报，也不是单纯头像，而是：

1. `人物是主角`
2. `场景和道具一起交代人格`
3. `颜色是分类信号`
4. `文字几乎不进图里`

所以如果我们想做得像，重点不是把字母做大，而是让用户看图就能猜出这是什么人。

## 2. 具体风格特征

### 2.1 构图

1. 画幅通常是接近正方形。
2. 角色是完整人物，不是只画头。
3. 角色不一定站正，往往带一点动作或场景互动。
4. 画面里经常有桌子、书、工具、门、植物、文件、道具等辅助元素。

### 2.2 造型

1. 扁平矢量感很强。
2. 没有明显描边，主要靠色块分层。
3. 四肢细长，比例略夸张，但不走Q版。
4. 脸部细节极少，重点在发型、姿态、服装和手里拿的东西。

### 2.3 配色

1. 每个类型有主色倾向。
2. 颜色通常是低饱和但不脏。
3. 有统一的肤色、灰色、白色、阴影色体系。
4. 背景通常比角色更淡，保证人物是焦点。

### 2.4 信息传达方式

MBTI 图的精华在于：

1. 不是靠文字解释“这个人是什么”。
2. 而是靠 `姿势 + 表情 + 道具 + 小场景` 让人秒懂。

例如：

1. 看书的人和桌上的文献，会让人联想到知识型。
2. 指挥动作和秩序型道具，会让人联想到管理型。
3. 活泼动作和装饰性物件，会让人联想到外放型。

## 3. 对我们产品的翻译

通用跑团版的插画应该遵守这套逻辑：

1. `侦查狗`
人物 + 放大镜/线索板/地面痕迹

2. `野蛮人`
人物 + 门板/武器/冲锋动作

3. `赌狗`
人物 + 骰子/手势/玄学氛围

4. `律师`
人物 + 规则书/文件/指示动作

5. `嘴遁王`
人物 + 对话气泡/手势/交涉状态

6. `社畜`
人物 + 清单/流程板/打工状态

7. `疑心病`
人物 + 可疑告示板/Among Us 风格小挂件/观察动作

也就是：

`不要把海报当人格图。`

`人格图应该更像一张“角色在做事”的小插画。`

## 4. 已落地到当前版本的改动

当前版本已经做了这些调整：

1. 从“字母+色带”的抽象海报，改为“角色+道具+场景”的扁平插画。
2. 图鉴、结果页、Top 3 卡片统一使用这套插画。
3. `SUSS` 特意加入了明显的 `Among Us` 风格小人玩偶式元素，作为跑团群聊梗点。
4. 导出 PNG 也同步嵌入了新插画，不再是旧版纯字卡海报。

## 5. 后续继续优化时的原则

1. 少堆文字，多堆动作和道具。
2. 一张图里只表达一个人格核心，不要贪多。
3. 配色保持低饱和统一，不要变成杂色梗图。
4. 允许有梗，但主体还是“人格插画”，不是“网络表情包拼贴”。
