# SBTI 研究笔记

## 1. 结论先行

`SBTI` 爆的不是“人格算法”，而是它把 `MBTI` 的可信外壳保留下来，再把结果命名和文案全面改造成了适合转发的抽象羞辱。

它做的不是“发明一种新测试”，而是：

1. 借用 MBTI 已经完成的用户教育。
2. 让用户继续相信自己正在做一个“有结构”的测试。
3. 在结果页用更有攻击性、更有互联网语感、更适合截图的标签完成收口。

## 2. SBTI 是如何模仿 MBTI 的

### 2.1 先模仿产品外壳

它非常清楚用户已经被 MBTI 训练过了，所以先保留这些要素：

1. `人格测试` 这个熟悉入口。
2. `多题作答 -> 输出一个固定人格` 的线性流程。
3. `人格代号 + 类型名` 的结果格式。
4. `维度模型 + 匹配算法` 的伪专业感。
5. `每种人格都有一段解释` 的阅读预期。

也就是说，SBTI 没有教育用户一个全新的交互，而是直接站在 MBTI 已经建立的“测试仪式感”上。

### 2.2 再模仿 MBTI 的“可信感”

从公开复刻版和分析资料看，SBTI 不是纯随机抽签，它有明确的数据结构：

1. 30 道左右题目。
2. 多个维度，且每个维度有对应题目。
3. 每道题 3 级选项。
4. 用维度向量去匹配固定类型。
5. 结果还带有隐藏人格和彩蛋机制。

这点非常关键：

如果它只是一个随机梗生成器，用户会笑，但未必会信。
如果它有一套看起来像“真的在测”的结构，用户就更容易接受那个羞辱式结果。

### 2.3 它连题目风格也在模仿 MBTI

SBTI 的题不是全都很抽象。它混了三层：

1. `正常人格题`
例如自我认知、边界感、关系依赖、规则感。
2. `生活情境题`
例如对象不回消息、遇到人际场景如何反应。
3. `梗化情境题`
例如窜稀、保温杯白酒、厕所、瞎选题。

这个配比很重要。

如果全是梗题，用户会觉得只是整活。
如果全是正经题，用户会觉得和 MBTI 没区别。
SBTI 的做法是：`用 70% 正常题保住“准感”，用 30% 抽象题保住“梗感”。`

## 3. SBTI 是如何做出自己风格化的

### 3.1 它没有继续做“理想化人格”

MBTI/16Personalities 的命名通常是：

1. 建筑师
2. 提倡者
3. 指挥官
4. 调停者

这些名字都偏“高地位、正向、职业化、可自我美化”。

SBTI 反过来做：

1. 死者
2. 废物
3. 小丑
4. 傻者
5. 狗屎人
6. 吗喽

这一步是它真正破圈的核心。

用户不是为了被赞美才转发，而是为了完成一次低风险的自嘲和认领。

### 3.2 它把“人格标签”改造成了“社会绰号”

MBTI 的类型像心理画像。
SBTI 的类型像朋友给你起的外号。

这会带来两个传播优势：

1. `更容易懂`
“死者”“废物”不需要理论解释。
2. `更容易转发`
外号比分析报告更适合截图。

### 3.3 它的文案不是解释，是表演

SBTI 的结果文案有几个高频手法：

1. `假装郑重`
大量“恭喜您”“极其罕见”“全球仅此一例”。
2. `故意夸张`
用宇宙、全人类、神秘定律等夸大结果。
3. `先骂后哄`
先给你一个攻击性标签，再告诉你“你其实不是这样，只是……”
4. `长段子化`
文案不是简洁说明，而是一段可朗读的互联网独白。
5. `高密度网络语感`
既有梗，也有点矫情，还有点装疯卖傻。

所以它的结果页本质上不是“人格报告”，而是一篇以你为主角的小作文。

### 3.4 它还模仿了 MBTI 的“分类快感”

MBTI 的快感之一，是把自己塞进 16 个盒子之一。

SBTI 保留了这件事，但把盒子换成了更适合年轻人互相点名的类别：

1. 你测出来是什么。
2. 你队友/同事/朋友测出来像哪个。
3. 你们群里谁最典。

这就把“测试”变成了“社交点名工具”。

## 4. 对我们最重要的启发

### 4.1 不要只学它的缩写，要学它的“严肃外壳 + 羞辱收口”

跑团版也要这样做：

1. 前台看起来像一个有结构的测试。
2. 后台真的有维度和匹配。
3. 但最终输出必须像车队里互相起外号。

### 4.2 不要再用“功能型、安全型”的人格名做结果主标签

像下面这类名字：

1. 线索吸尘器
2. 过本发动机
3. 规则说明书

它们不是不能用，但更像产品经理内部术语。
不够像朋友会拿来点名你的外号。

SBTI 风格要求主标签更像：

1. 刁民
2. 律师
3. 野蛮人
4. 社畜
5. 赌狗
6. 杠精

也就是 `社会角色 / 攻击性外号 / 低地位标签`。

### 4.3 跑团版结果必须同时满足三点

1. `桌内能懂`
玩过团的人一眼知道在说谁。
2. `桌外也能懂`
没深玩过的人至少能从名字里感受到人设。
3. `骂得不重但够狠`
用户愿意自嘲，不会觉得被真正羞辱。

### 4.4 结果命名最好的公式

推荐使用：

`社会绰号 / 职业标签 / 网络污名词 + 跑团行为`

例如：

1. `律师`：规则律师，抠字眼，纠流程。
2. `刁民`：不信NPC，不信KP，不信“普通房间”。
3. `野蛮人`：先砍再说，先踹再聊。
4. `社畜`：负责收拾队友烂摊子，把主线硬拖回去。
5. `赌狗`：关键时刻和骰子谈宗教。
6. `戏精`：台词一开口，全桌都得看你演。
7. `杠精`：任何一句“理论上不行”都能激活你。

## 5. 我们的跑团版命名方向

基于上面的研究，当前版本的人格命名应该从“功能性标签”转向“绰号型标签”。

新版 16 个正式人格建议为：

1. `CLUE` 侦查狗
2. `BARB` 野蛮人
3. `ROLL` 赌狗
4. `ROLE` 戏精
5. `MEME` 乐子人
6. `HIDE` 苟王
7. `RULE` 律师
8. `TALK` 嘴遁王
9. `LORE` 设定怪
10. `PAIN` 苦主
11. `SUSS` 疑心病
12. `TRAP` 碰瓷王
13. `WORK` 社畜
14. `JOKE` 小丑
15. `KANG` 杠精
16. `SCAN` 刁民

这里的逻辑是：

1. `代号` 继续保留测试系统感。
2. `中文名` 变成群里会直接拿来叫人的词。
3. `结果描述` 从解释功能，改成表演一种团桌身份。

## 6. 产品层面的落地原则

### 题目

题目不需要全部改成疯话，但以后应遵守这个比例：

1. 60% 跑团日常题
2. 25% 跑团冲突/出事题
3. 15% 轻度抽象/发病题

### 结果页

结果页要更像一张“团桌通缉令”而不是一张“人格分析报告”。

必须有：

1. 大标签
2. 一句暴击
3. 一段风格化长描述
4. 三条典型行为
5. 一句发群文案

### 分享逻辑

跑团版最好的分享目标，不是朋友圈，是：

1. 开团群
2. 复盘群
3. 车卡群
4. 跑团好友私聊

所以分享文案要天然适合一句话扔群里：

`我测出来是律师，你们几个谁测都别想跑。`

## 7. 一句话总结

`SBTI` 并不是把 MBTI 做得更专业，而是把 MBTI 做得更像一个大家愿意认领的外号系统。

跑团版如果想真有传播力，也必须从“功能描述”升级到“外号羞辱”。
