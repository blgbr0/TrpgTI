# COC 版本说明

`coc.html` 是一个独立于通用版的人格测试页面，目标不是做“通用跑团梗的轻微偏科”，而是直接把文案、题面、隐藏人格都拉到 `COC` 黑话层级。

## 版本特点

1. 题目全部偏向 `侦查 / 聆听 / 图书馆 / 掉san / 暗投 / 撕卡 / 坏KP`。
2. 人格标签更像 COC 固桌里的常驻病症。
3. 隐藏人格直接围绕 `SAN0 / 撕卡侠 / 坏KP受害者` 触发。
