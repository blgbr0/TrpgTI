# 跑团梗库

这份清单用于给人格命名、题面语气、结果页文案做统一校准。

## 1. 这轮主要参考到的高频跑团梗

基于搜索结果和中文跑团圈常见表达，当前最稳定、最容易被认领的梗主要集中在这几类：

1. `KP / PL`
主持人与玩家的基本关系梗，适合做“坏KP”“刁民”“律师”“杠精”类结果。

2. `侦查 / 聆听`
中文 COC 圈最有群众基础的动作梗之一，天然适合“侦查狗”“刁民”“疑心病”。

3. `大成功 / 大失败 / d100`
最适合做命运感和事故感的梗，直接支撑“赌狗”“碰瓷王”。

4. `车卡`
人物卡准备与拖延是稳定笑点，适合“白板怪”。

5. `撕卡`
跑团里最强烈的损失感表达之一，适合“苟王”“苦主”“赌狗”。

6. `SAN / 掉san`
COC 圈最出圈的精神伤害梗，适合“苦主”“疑心病”，也非常适合做隐藏人格 `SAN0`。

7. `坏KP`
群聊和梗图里常见，适合结果页把锅推给主持人，增强认领感。

8. `团灭`
适合事故型人格，但首版主结果里更适合放在文案里，而不是只做结果名。

9. `鸽 / 迟到`
开团前的社交梗，适合隐藏人格“鸽子精”。

10. `舔包 / 捡东西 / 刮地皮`
场景搜刮与资源占有欲，是非常适合做彩蛋羞辱的梗。

11. `嘴遁`
比“销冠”更跑团圈，适合结果名直接使用，例如“嘴遁王”。

12. `乐子人`
虽然不是跑团专属，但和整活、作死、群聊传播天然适配。

## 2. 为什么这些梗适合做结果名

这些梗有三个共同点：

1. 玩过两三次团的人就能懂。
2. 放到群里点名别人也成立。
3. 天然带点轻攻击性，但不至于真伤人。

所以它们非常适合改造成：

- 主结果标签
- 隐藏人格
- 分享图短句
- 彩蛋题
