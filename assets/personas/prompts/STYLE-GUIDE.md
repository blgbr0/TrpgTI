# 统一风格说明

以下 16 张图都建议遵循同一套插画体系，这样图鉴页和结果页会更像同一个产品。

## 统一要求

- 竖版人格测试插画，比例 `4:5`
- 气质参考高传播人格测试插画：亲和、清晰、可截图、角色识别度高
- 不要直接复刻现成 MBTI 角色，保持“受启发但不是复写”
- 风格建议：`editorial illustration + soft 3D + flat shape + clean rim light`
- 人物要有清晰职业轮廓、手势动作和能一眼识别的人设道具
- 背景不要复杂叙事，更适合做结果页封面
- 色彩偏暖金、深墨绿、旧纸金属、低饱和宝石色
- 角色表情夸张一点，利于社交传播

## 统一负面约束

- 不要纯写实摄影
- 不要二次元厚涂大眼风
- 不要赛博霓虹廉价紫粉
- 不要过度阴暗恐怖，整体仍然要像“能发群”的人格测试插画
- 不要塞太多背景小物导致主体不清
